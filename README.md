This repository contains comprehensive Kubernetes configuration files and project setups used for learning, experimentation, and real-world DevOps workflows.
It includes:

  - Pod, Deployment, DaemonSet, and StatefulSet examples
  - Service (ClusterIP, NodePort, LoadBalancer) and Ingress configurations
  - ConfigMaps, Secrets, Probes, and Resource Limits
  - Persistent Volumes & Persistent Volume Claims
  - Helm charts and reusable templates
  - Monitoring stack (Prometheus, Grafana)
  - GitOps-style deployments
  - Best practices for production-ready Kubernetes

This repo acts as a complete reference for managing containerized workloads efficiently on Kubernetes.
